🔄 核心数据流

AI 调用流

客户端 → API网关(JWT) → 路由引擎 → 模型适配器 → LLM → 安全过滤器 → 计费引擎 → 返回响应

安全扫描流

资产变更事件 → NotifyAssetChange → 10s防抖 → 5扫描器并行 → 结果同步(知识库/安全日志)

预算管控流

API请求 → Redis Lua原子检查 → 3级阈值判断 → pass/alert/block → Agent Guard异常检测

知识库安全流

文件上传 → 创建条目(pending_scan) → SecurityAgent扫描 → 更新状态(safe/risky/blocked)
